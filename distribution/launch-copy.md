## Show HN Draft

Title:

`Show HN: Prompt Table Cleaner - paste messy tables, get Markdown and JSON for AI prompts`

Body:

I built a small static demo for a problem I hit when working with AI tools: copied tables often lose structure before they reach the prompt.

Prompt Table Cleaner takes CSV, TSV, or copied table-like text and returns:

- Markdown table
- JSON rows
- compact prompt summary

The repo is intentionally small. I am trying to learn which workflow people actually want before building a full product: Chrome extension, clipboard tool, CLI, live HTML table extraction, or agent integration.

Demo is in `demo/index.html`. GitHub Pages entry is in `docs/index.html`.

Repo:

`https://github.com/yitengruntu/prompt-table-cleaner`
