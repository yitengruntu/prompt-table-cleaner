# Scope

## Working Name

Prompt Table Cleaner

## Core Job

Turn copied table-like text into clean, prompt-ready Markdown and JSON.

## Target User

People who regularly paste structured data into AI tools.

## First Test Surface

GitHub repo with static demo and sample input.

## Explicit Non-Goals

- No paid checkout.
- No extension build.
- No remote server.
- No upload of private data.
- No background clipboard watcher.

## Continue Rule

Build a real workflow only after external users ask for a concrete form factor: extension, CLI, browser bookmarklet, or clipboard tool.
