# Validation Plan

## Hypothesis

AI-heavy users lose time cleaning copied tables before putting them into prompts. A small tool that converts table-like text into Markdown and JSON may earn attention if the workflow pain is real.

## Channels

- GitHub-first public probe.
- Reddit feedback post in `r/PromptEngineering`.

## Public Links

- Demo: https://yitengruntu.github.io/prompt-table-cleaner/
- Repo: https://github.com/yitengruntu/prompt-table-cleaner
- Reddit post: https://www.reddit.com/r/PromptEngineering/comments/1ug16w8/i_made_a_tiny_tool_to_clean_copied_tables_into/

## Window

- First read: 24 hours after the Reddit post.
- Decision read: 72 hours after the Reddit post.

## Leading Signals

- GitHub views or clones
- stars or forks
- issue asking for a specific workflow
- direct comment asking for extension, CLI, or clipboard support
- Reddit comments with specific source tables, output formats, or repeat-use workflows

## Continue Condition

At least 2 concrete workflow requests or 5 technical interactions across GitHub and Reddit.

## Kill Condition

No stars, forks, issues, comments, or specific workflow requests after the 72-hour public test.
